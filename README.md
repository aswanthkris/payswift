# PaySwift

PaySwift is a modern fintech application designed to simplify digital payments and wallet management. It features a sleek user interface, real-time transaction tracking, and insightful financial analytics.

## Features

- **User Authentication**: Secure login and signup with JWT.
- **Wallet Management**: View balance, add money, and transfer funds.
- **Transaction History**: Detailed record of all incoming and outgoing transactions.
- **Analytics**: Visual charts for income vs. expense and daily spending trends.
- **Beneficiary Management**: Save and manage frequent contacts.
- **Modern UI**: Responsive design built with Tailwind CSS and Framer Motion.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Recharts, Zustand.
- **Backend**: Node.js, Express, PostgreSQL, Sequelize.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- PostgreSQL

### Backend Setup

1.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Create a `.env` file based on your configuration (DB credentials, JWT secret).
4.  Start the server:
    ```bash
    npm run dev
    ```

### Frontend Setup

1.  Navigate to the `Frontend` directory:
    ```bash
    cd Frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
